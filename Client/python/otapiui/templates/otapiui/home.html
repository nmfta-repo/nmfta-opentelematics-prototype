Prototype implementation of the 0.1-rc.4 version of the Open Telematics API. This GUI client, developed in python is connecting to a server C# implementation of OTAPI.

The following OTAPI endpoints are used in this demo:

  • health/current
  • fleet/locations/latest
  • fleet/faults/feed
  • event_logs/feed
  • export/vehiclerecords/status
  • export/allrecords/status
  • i18n
